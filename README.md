# Rune OS - Fenrir

A customized Ubuntu 24.04 LTS desktop build focused on system optimization and clean workspace tweaks. This project is currently in the early development phase.

## Current State
The focus right now is ensuring the core desktop environment boots properly and stays rock-solid before applying deeper system modifications and custom toolsets.

* Base: Ubuntu 24.04 LTS (Noble Numbat)
* Architecture: x86_64
* Build Tool: Cubic

## Development Plan
- Configure stable vanilla boot environment
- Replace default branding, logos, and Plymouth themes
- Apply custom kernel and performance tweaks
- Inject specific system optimization tools
